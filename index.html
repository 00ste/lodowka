Lodówka

About Lodówka

  • Lodówka (IPA: [lɔˈdufka], "fridge" in Polish) is a simple web application that helps you keep track of the expiration date of products you bought.
  • All the data is stored in the local storage of your browser, and the application will not work if the browser erases the browsing data.
  • You can download a backup of your data as a JSON file, and import this file on another device or in case the browser data is lost.
